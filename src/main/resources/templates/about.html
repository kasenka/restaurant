В рамках проекта было разработано веб-приложение с использованием современных технологий разработки.


Начало работы над проектом - 01.03.2025

Окончание работы над проектом - 20.05.2025

Использованный стек технологий:

  • Backend: Java, Spring Boot
  • Frontend: HTML, CSS, Thymeleaf
  • База данных: PostgreSQL

Опыт, полученный при разработке проекта:

    Backend (Java + Spring Boot):

  • Разработано монолитное приложение на Java и Spring Boot, на базе архитектуры MVC
  • Реализована обработка запросов и форм
  • Model для передачи данных в шаблоны
  • Работа с Spring Security – настройка аутентификации и авторизации
  • Работа с базами данных – подключение PostgreSQL, использование Spring Data JPA (@Entity, @Repository), написание SQL-запросов
  • Обработка ошибок и проверка данных – настройка @ExceptionHandler

    • Frontend (HTML, CSS, Thymeleaf):

  • Динамические страницы – использование Thymeleaf (th:text, th:each, th:if) для рендеринга данных из Java
  • Верстка и стилизация – создание адаптивных интерфейсов с HTML/CSS
  • Формы и валидация – передача данных из фронтенда в бэкенд
  • Интеграция с Spring – безопасная подстановка данных через th:value, работа с Model
  • Работы с HTML и CSS для создания адаптивного и визуально привлекательного интерфейса, включая стилизацию форм и отображение данных
  • Использованы общие шаблоны с помощью th:fragment, что позволило повторно использовать элементы интерфейса (хедер и футер)

    • База данных (PostgreSQL):

  • Проектирование схемы БД – создание таблиц, связей
  • Интеграция PostgreSQL со Spring Data JPA для упрощения взаимодействия с базой
  • CRUD-операции (создание, чтение, обновление, удаление) для работы с данными пользователей и других сущностей
